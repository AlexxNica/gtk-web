GTK+-2.6

    The current last stable version of GTK+ is 2.6. This is the last version of GTK+ that does not require cairo.

    Source

    You'll need to get the GLib, GTK+, Pango, and ATK packages from the above locations. You may also need some of the external dependencies that are mirrored in the dependencies subdirectory.

GTK+-2.4

    The next older stable version of GTK+ was 2.4. Since GTK+ 2.6 and 2.8 have full source and binary compatibility with GTK+ 2.4, there should be no reason to install GTK+-2.4 at this point; if you need it, for some reason, it is available from:

    Source

    You'll need to get the GLib, GTK+, Pango, and ATK packages from the above locations. You may also need some of the external dependencies that are mirrored in the dependencies subdirectory.

    Installation guide

GTK+-1.2

    Some applications still require GTK+-1.2, an older stable version of GTK+. You can have the runtime and development environments for both GTK+-2.4 and GTK+-1.2 installed simultaneously on your computer.

    Source

    You'll need to get and install both the GLib and GTK+ packages from the above locations.

CVS

    The latest source is always available via anonymous CVS. The GLib and GTK+ sources are hosted in GNOME's CVS repository as the 'glib' and 'gtk+' CVS modules. Instructions on using GNOME's CVS repository.
