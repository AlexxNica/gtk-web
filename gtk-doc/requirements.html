Requirements

Basic requirements:

  • Perl - the main scripts are in Perl.

For XML output (recommended):

  • The DocBook XML DTD.
  • The DocBook XSL Stylesheets.
  • libxslt & libxml2.

For SGML output (not actively maintained any more):

  • The DocBook SGML DTD.
  • The DocBook DSSSL Stylesheets.
  • Jade or OpenJade.

Most distributions now have packages for all of these, so I would strongly advise that you grab those.
