GTK+-2.8 Plans

Overview

    It is not clear yet which features will be done in the timeframe for 2.8, but possible "major themes" for the 2.8 release are

    • Cairo support
    • RGBA visuals
    • Printing

    These features are explained in detail in Owens talk on the future of rendering in GNOME

    Some things slipped from 2.6:

    • Rotated text in Pango (#99468)

    • Rotated text in labels

    • Better language binding support - full introspection, etc.

Tentative schedule

    No schedule yet. The timeframe for the 2.8 release is mid 2005.
