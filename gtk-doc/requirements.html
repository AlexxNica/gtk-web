XML Generation

For writing XML files you will need these packages. This is highly recommended!

  • DocBook XML DTD.
  • DocBook XSL Stylesheets.
  • libxslt & libxml2.

SGML Generation

For writing SGML files you will need these packages. This is not actively maintained any more:

  • DocBook SGML DTD.
  • DocBook DSSSL Stylesheets.
  • Jade or OpenJade.

Most distributions now have packages for all of these, so it is strongly advised that you grab those if possible.
