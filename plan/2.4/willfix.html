GTK+-2.4 Bugs to Fix

    The following list is bugs that are scheduled to be fixed for GTK+-2.4:

    • Deadline for adding issues to it is this Friday, Feb 20 2004. (important new issues that are found after Friday can be added as exceptions)
    • Issues on the list should generally either be blockers for the release or bugs with patches
    • Things not on the list will generally not be fixed

GLib

  • #63621: Add GAtomicInt for atomic integer operations. API for atomic operations with efficient implementations for all major platforms, including tests.
  • #78414: g_queue_remove() and g_queue_remove_all(). A full GList/GSList equivalent API set for GQueue, including tests.
  • #118536: Make g_signal_connect_object'ed handlers disconnect when the data object is destroyed.

Pango

  • #130661: efence array overrun detection. Memory corruption bug with proposed fix.

GDK

  • #119804: Hide -lXcursor dependency Otherwise Xcursor will end up in the ABI

gdk-pixbuf

  • #134235: Crush gdk_pixbuf_new_from_file. A crash upon loading certain bmp images.

GtkFileChooser

  • #132894: gtkfilechooserdefault->local_only is never used. This is not implemented at all.
  • #128396: GtkFileChooser doesn't show any files initially if filters are used.
  • #134294: eog crash on selecting 'Home' in Load image dialog.
  • #134028: Focus sequence is wrong. Pressing Tab inside the file chooser doesn't yield a reasonable focus order.
  • #134409: FileChooser bookmarks buttons. Spacing is wrong and it is not obvious what they apply to.
  • #134053: gtk_file_chooser_set_current_folder leak.
  • #134457: Width of the file selector window increases continuously. With long paths, the current folder label gets too wide.
  • #114065: Bad warning on G_BROKEN_FILENAMES. Needs a string change.

GtkComboBox, GtkComboBoxEntry and GtkEntryCompletion

  • #131542: GtkEntryCompletion crashes on "row_inserted" signal.
  • #127430: Conflict between entry/completion key bindings.
  • #127623: GtkComboBox leaking widgets.
  • #127804: GtkComboBoxEntry critical warnings.
  • #132847: GtkComboBox widget is not accessible.
  • #134373: strange behaviour of url entry.
  • #134468: GtkComboBoxEntry Windows mode keynav issues.
  • #134470: GtkComboBox focus issues.
  • #134471: GtkComboBox in Windows mode misses a grab.
  • #134722: Crashes in libgobject when entering items for a GtkComboBox in glade-2.

GtkTextView

  • #78513: Scrolling by keeping Page Up/Down down is broken. Not sure this can be analysed in time for 2.4.

Other GTK+

  • #82099: GtkImageMenuItem needs a style property or GtkSetting to toggle showing images. Adds settings to suppress button and menuitem images.
  • #112647: Add indicator-size style property for checkmenuitem. Important for the large-print themes.
