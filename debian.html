Debian Packages

Debian GNU/Linux binary and source packages are available for GTK+ 1.0 and 1.2, and have always been an official part of Debian Linux.

The official URL for GTK+ 1.2 Debian packages, which is automatically in synch with the Debian FTP archives, is http://packages.debian.org/libgtk1.2.

If you need older GTK+ 1.0 Debian packages, you can fetch them from http://packages.debian.org/libgtk1.

Make sure you use packages from "stable" if you are running Debian GNU/Linux version 2.1 (codename: slink) and from "unstable" if you are running version 2.2 (codename: potato).
