GTK+-2.0

    The current last stable version of GTK+ was 2.0. Since GTK+-2.2 has full source and binary compatibility with GTK+-2.0, there should be no reason to install GTK+-2.0 at this point; if you need it, for some reason, it is available from:

    Source

GTK+-1.2

    Many applications still require GTK+-1.2, an older stable version of GTK+. You can have the runtime and development environments for both GTK+-2.2 and GTK+-1.2 installed simultaneously on your computer.

    Source
    Binary

    You'll need to get and install both the GLib and GTK+ packages from the above locations.

CVS

    The latest source is always available via anonymous CVS. The GLib and GTK+ sources are hosted in GNOME's CVS repository as the 'glib' and 'gtk+' CVS modules. Instructions on using GNOME's CVS repository.
