GTK+-2.4 Plans

Overview

    GTK+-2.4 will be a release maintaining source and binary compatibility with GTK+-2.0 and GTK+-2.2 while introducing new features, like a better file selector and replacement "combo box" widget. It will be a time based release; that is, we'll pick a timescale and base what goes in on that rather than vice-versa.

Tentative schedule

    A timescale of 8-9 months seems reasonable; that is, a final 2.4.0 in late August or early September. A rough schedule for a September 1 release is:

    28 December 2002 Start of development
    1 February 2003 Have proposals for all major features, decide what goes in, doesn't
    1 June 2003 All major features are in
    1 July 2003 Feature freeze
    1 August 2003 API/ABI freeze
    1 September 2003 2.4.0

Possible features

  • File selector (we have to get this one) (#29087)

  • Combo widget (Kris has made progress here) (#50554)

  • New action-based menu API (based on James Henstridge's work, most likely. #55393)

  • Toolbar improvements (See EggToolbar in libegg. #55393)

  • GObject private data (#101959)

  • Properties for interfaces (#105894)

  • Full Unicode 3.2 (4.0?) support, including non-BMP portions. (#68435, #101081)

  • GtkModelFilter (Pretty much done in libegg). (#60219)

  • Make RTL text editing really work (automatic paragraph direction, etc., #70451, #70399)

  • Height-for-width geometry management. (Define a new interface, support it GtkVBox, GtkLabel would be the minimum. #104188)

  • Reworked tooltips (#50619, #56291, #62174, #65490)

  • Autocompletion and history for GtkEntry (#69613)

  • Disclosure triangle widget (an implementation by Iain Holmes can be found in gnome-panel/gnome-panel/disclosure-widget.[ch]. #60553)

  • GnomeColorPicker style selection buttons for colors, fonts, etc

  • Optimized pixbuf operations -- scaling down a lot (#80925), unscaled ops (#80927)

  • "Smooth scrolling", like in Internet Explorer (#103811)

  • GtkResizeGrip -- a widget implementing a resize grip. (#73359)

  • Integration of directfb port.

  • XCursor support for GDK. (#69436)
