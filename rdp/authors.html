GTK+ 2.0 Development

In GTK+ 2.0.x the reference documentation has been integrated with the GLib and GTK+ packages (i.e. in the glib, gtk+ and pango modules in GNOME's CVS repository). The GTK+ developers have finally realized the importance of documentation and now the rule is that no new code is allowed to go into GTK+ without inline documentation for all of the functions. Great!

However, there are still many parts of GTK+ and GLib with little or no documentation. Hopefully we will soon devise a procedure for non-developers to help out by submitting documentation for these parts. Watch this space!

Is there a mailing list?

Yes. Please subscribe to it if you want to help with the documentation.

To subscribe send a message to gtk-doc-list-request@redhat.com with "subscribe" as the Subject.
