Related Projects

GDK backends

    GTK+ for Win32 - this backend is included in the regular GTK+ packages.

    GTK+ on DirectFB - this backend will likely be included in GTK+ 2.6.

    GTK+ for BeOS - this project is dead, but the code may still be of interest.

    GTK+OSX - a project porting GTK+ 1.2 to OS X.

    Gtk Mac OS X - a port of GTK+ to OS X. This port will be part of GTK+ 2.10.

Input methods

    gtk-im-extra - a collection of third party GTK+ input method modules.

    imime - An immodule for GTK+2 using Windows IME. This input method has been integrated in GTK+ 2.6

gdk-pixbuf image loaders

    librsvg - an SVG rendering library which includes an gdk-pixbuf loader for SVG images.

    libwmf - a library to interpret the Windows Metafile format wmf which includes a gdk-pixbuf loader.

    swfdec - a library for rendering Flash® animations and games which includes a gdk-pixbuf loader.

    GStreamer - has gdk-pixbuf loader capable of loading AVI and mpeg animations.

    SLgtk - has a gdk-pixbuf loader for the FITS image format, which is heavily used within Astronomy.

Other projects which are related to GTK+ in some way

    GTK-Doc - the documentation tool used for GLib, GTK+, Pango and many other parts of the Gnome library stack

    Pango - the internationalized text handling framework used in GTK+.

    Cairo - a vector graphics library

    GNOME - a complete desktop environment built upon GTK+.

    GtkGLExt - an OpenGL extension to GTK 2.0 or later.

    PangoPDF - implements a version of the Pango library with a PDF backend for creating PDF output.

    GTK-Wimp - a theme engine for GTK+ which uses the native drawing functions on Win32. This theme engine has been integrated in GTK+ 2.6
