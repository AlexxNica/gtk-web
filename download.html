Unstable release

You can check out the latest unstable release of GTK+ and GLib using Subversion.

svn checkout http://svn.gnome.org/svn/gtk+/trunk gtk
svn checkout http://svn.gnome.org/svn/glib/trunk glib

For more information on this, see the instructions on how to use the repository
